Validador Biométrico

Demo del modo declarativo: un integrador agrega 1 script + 1 botón a su página y obtiene validación biométrica completa con QR, polling y webhook firmado HMAC. Sin escribir JavaScript.

Esto es todo el código del frontend

<script src="https://validador-atm.duckdns.org/embed/embed-v1.js" defer></script>

<button data-validador-biometrico
        data-session-url="/api/crear-sesion"
        data-on-approved="mostrarResultado">
  Validar identidad
</button>

<script>
  window.mostrarResultado = async (detail) => {
    // detail = { session_id, completed_at }
    // Para mostrar PII en la UI, fetch al backend del integrador:
